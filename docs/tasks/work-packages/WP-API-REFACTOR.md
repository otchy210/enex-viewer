# WP-API-REFACTOR

## 対象タスク
- [x] T-019

## 目的
API 入力解析の重複コードを安全に共通化し、バリデーション品質を統一する。

## 成果物
- 共通入力解析ヘルパー（query/params）
- 既存 controller での再利用
- 既存仕様を維持する回帰テスト

## 完了条件
- `npm run typecheck -w apps/api` が通過
- `npm run test:api` が通過
- 配列クエリ/不正数値の扱いが意図通りに統一される

## 更新ルール
- このワークパッケージを担当したエージェントは、担当タスク完了時に該当チェックボックスを `[x]` に更新する。
